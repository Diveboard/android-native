Eclipse projects. Here's how the projects were restructured:

In FacebookSDK:
* AndroidManifest.xml => facebookSDK\src\main\AndroidManifest.xml
* res\ => facebookSDK\src\main\res\
* src\ => facebookSDK\src\main\java\
In PagerSlidingTabStrip:
* AndroidManifest.xml => pagerSlidingTabStrip\src\main\AndroidManifest.xml
* lint.xml => pagerSlidingTabStrip\lint.xml
* res\ => pagerSlidingTabStrip\src\main\res\
* src\ => pagerSlidingTabStrip\src\main\java\
In UserVoiceSDK:
* AndroidManifest.xml => userVoiceSDK\src\main\AndroidManifest.xml
* libs\android-support-v13.jar => userVoiceSDK\libs\android-support-v13.jar
* libs\commons-codec-1.8.jar => userVoiceSDK\libs\commons-codec-1.8.jar
* libs\signpost-commonshttp4-1.2.1.2.jar => userVoiceSDK\libs\signpost-commonshttp4-1.2.1.2.jar
* libs\signpost-core-1.2.1.2.jar => userVoiceSDK\libs\signpost-core-1.2.1.2.jar
* lint.xml => userVoiceSDK\lint.xml
* res\ => userVoiceSDK\src\main\res\
* src\ => userVoiceSDK\src\main\java\
In Diveboard:
* AndroidManifest.xml => diveboard\src\main\AndroidManifest.xml
* assets\ => diveboard\src\main\assets\
* libs\android-support-v13.jar => diveboard\libs\android-support-v13.jar
* libs\bugsense-3.6.jar => diveboard\libs\bugsense-3.6.jar
* libs\commons-codec-1.8.jar => diveboard\libs\commons-codec-1.8.jar
* libs\commons-logging-1.1.3.jar => diveboard\libs\commons-logging-1.1.3.jar
* libs\fluent-hc-4.3.2.jar => diveboard\libs\fluent-hc-4.3.2.jar
* libs\httpclient-4.3.2.jar => diveboard\libs\httpclient-4.3.2.jar
* libs\httpclient-cache-4.3.2.jar => diveboard\libs\httpclient-cache-4.3.2.jar
* libs\httpcore-4.3.1.jar => diveboard\libs\httpcore-4.3.1.jar
* libs\httpmime-4.3.2.jar => diveboard\libs\httpmime-4.3.2.jar
* libs\libGoogleAnalyticsServices.jar => diveboard\libs\libGoogleAnalyticsServices.jar
* libs\signpost-commonshttp4-1.2.1.2.jar => diveboard\libs\signpost-commonshttp4-1.2.1.2.jar
* libs\signpost-core-1.2.1.2.jar => diveboard\libs\signpost-core-1.2.1.2.jar
* lint.xml => diveboard\lint.xml
* res\ => diveboard\src\main\res\
* src\ => diveboard\src\main\java\
